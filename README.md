# Smart Fridge and Culinary Assistant

A smart web application that helps you manage your refrigerator items and discover recipes based on available ingredients.


## Features

- **Inventory Management**: Keep track of items in your fridge
- **Recipe Discovery**: Find recipes based on your available ingredients
- **Smart Suggestions**: Get personalized recipe recommendations

## Installation

1. Clone the repository:
```bash
git clone https://github.com/MrJash/smart-fridge-and-culinary-assistant.git
cd smart-fridge-and-culinary-assistant
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```

## Usage

The application is ready to use immediately after startup. No login is required to access the core features of the Smart Fridge and Culinary Assistant. Simply launch the application and begin managing your inventory and discovering recipes.

### Getting Started

1. Add ingredients to your fridge inventory
2. Upload photo of your fridge inventory to the Web App
3. Browse available recipes based on what you have
4. View detailed instructions and nutrition information
